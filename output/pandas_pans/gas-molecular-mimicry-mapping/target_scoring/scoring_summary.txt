MMPred Integration Note
-----------------------
MMPred (Junet et al., Front Genet 2024;15:1500684, PMID 39722794) provides
MHC-II molecular mimicry prediction via CNN + NetMHCIIpan.  The full tool
requires TensorFlow 2.6, BLAST+, and NetMHCIIpan 4.1 (academic license),
exceeding the 8GB RAM constraint.  A lightweight reimplementation using
TEPITOPEpan 9-pocket binding profiles with percentile-rank calibration
(10,000 random peptides/allele) was used instead.

*03:01, *11:01, *13:01 — expanding from the 3-allele PSSM predictions.
